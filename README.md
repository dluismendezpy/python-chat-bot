#Koly
**Chatbot using python3**

#### Setup on linux
    git clone https://github.com/dluismendezpy/python-chat-bot.git
    cd python-chat-bot
    virtualenv env 
    source env/bin/activate
    pip3 install -r requirements.txt
    python3 main.py

#### Setup on Windows
    git clone https://github.com/dluismendezpy/python-chat-bot.git
    cd python-chat-bot
    python -m venv env
    env\Scripts\activate.bat
    pip install -r requirements.txt
    python main.py

#### Setup on MacOS
    git clone https://github.com/dluismendezpy/python-chat-bot.git
    cd python-chat-bot
    virtualenv env 
    source env/bin/activate
    pip install -r requirements.txt
    python main.py
